Privacy Policy

Last updated: March 2026

1. Introduction

Welcome to Code Collab. We respect your privacy and are committed to protecting it. This Privacy Policy explains how we collect, use, and safeguard your information when you use our real-time collaborative code editor.

2. Information We Collect

No Accounts Required: Code Collab is designed to be frictionless. We do not require you to create an account, provide an email address, or set a password to use the core features of the platform.

Code and Room Data: When you create or join a room, the code you write, the chat messages you send, and the temporary display name you choose are processed by our servers to enable real-time collaboration. This data is transient and is only kept in active memory while the room is actively being used.

3. How We Use Your Information

We use the data generated during your session strictly to provide the service: broadcasting code changes, routing chat messages, and allowing the AI Assistant to generate responses based on your prompts. We do not use your code to train our AI models.

4. Data Sharing and Security

We do not sell, trade, or rent your information to third parties. Rooms are secured via unique, hard-to-guess URL hashes. While we prioritize security, please be aware that rooms are technically public to anyone who possesses the exact link. We recommend not pasting highly sensitive credentials or proprietary code into temporary rooms.

5. Changes to This Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page.

If you have any questions about this Privacy Policy, please contact us via our GitHub repository.
